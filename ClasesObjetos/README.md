# Conceptos de la POO orientada a objetos en Python

Este modulo cubre los pilares fundamentales de la POO en Python: 
* **Encapsulamiento:** Restricción de acceso mediante atributos privados/protegidos y control de estado a través de los métodos get y set para la lectura y modificación de los atributos de una clase
* **Herencia y herencia múltiple**
* **Método Dunder:** Inicialización de estados y constructores utilizando el método '__init__'

## Ejercicios
* **persona.py:** Modelado básico de una clase, instanciación de objetos y paso de parámetros al constructor
* **coche.py:** Implementación práctica de la encapsulación con atributos privados y protegidos y de los métodos get y set
